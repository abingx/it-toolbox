# Changelog - v2.2.0

**发布日期**: 2026-03-24

IT Toolbox v2.2.0 从 v2.1.0 的 106 个工具扩展到 116 款工具，新增 10 个实用工具。

---

## 新增工具（10个）

### 宽高比计算
- 输入宽度和高度自动计算宽高比
- 支持简化比例显示（如 16:9）
- 内置常见分辨率参考列表
- 支持自定义尺寸计算
- 一键复制计算结果

### CSS 单位换算
- 支持 px/rem/em/vw/vh/pt/pc/in/cm/mm 单位互转
- 可配置根字体大小（默认 16px）
- 可配置视口宽度
- 实时显示全部单位换算结果
- 一键复制换算结果

### 罗马数字转换
- 阿拉伯数字转罗马数字（支持 1-3999）
- 罗马数字转阿拉伯数字
- 输入验证和错误提示
- 一键复制转换结果

### 常用端口参考
- 收录 50+ 常用服务端口
- 按协议分类（HTTP/FTP/邮件/数据库/消息队列/开发工具等）
- 支持关键词搜索过滤
- 显示端口说明和服务名称
- 一键复制端口号

### 邮箱格式验证
- RFC 5322 标准格式验证
- 检测常见邮箱域名拼写错误
- 支持批量验证（每行一个邮箱）
- 显示验证结果详情
- 一键复制有效邮箱列表

### 质数检测
- 检测单个数字是否为质数
- 质因数分解功能
- 生成指定范围内的质数序列
- 支持大数计算
- 一键复制结果

### GCD/LCM 计算
- 最大公约数（GCD）计算
- 最小公倍数（LCM）计算
- 支持多个数字同时计算
- 显示计算过程
- 一键复制结果

### WHOIS 查询
- 查询域名注册信息
- 显示注册商、注册日期、过期日期
- 显示域名状态
- 显示 DNS 服务器
- 支持常见域名后缀

### SSL 证书检测
- 检测域名 SSL 证书状态
- 显示证书颁发机构
- 显示证书有效期
- 显示证书链信息
- 检测证书是否过期

### HTTP 安全头检测
- 检测目标 URL 的安全头配置
- 检测项目包括：
  - Strict-Transport-Security (HSTS)
  - Content-Security-Policy (CSP)
  - X-Frame-Options
  - X-Content-Type-Options
  - X-XSS-Protection
  - Referrer-Policy
  - Permissions-Policy
- 显示安全评分和建议

## 技术亮点

### 纯前端工具（7个）
以下工具采用纯前端实现，无需后端 API 支持：
- aspect-ratio（宽高比计算）
- css-unit-convert（CSS单位换算）
- roman-numeral（罗马数字转换）
- port-reference（常用端口参考）
- email-validate（邮箱格式验证）
- prime-checker（质数检测）
- gcd-lcm（GCD/LCM计算）

### API 依赖工具（3个）
以下工具需要后端 API 支持：
- whois-lookup（WHOIS查询）
- ssl-checker（SSL证书检测）
- headers-check（HTTP安全头检测）

### 功能特色
- port-reference 收录 50+ 常用端口定义，支持搜索过滤
- prime-checker 支持质因数分解和质数序列生成
- email-validate 支持 RFC 5322 标准验证和批量处理
- 所有工具通过 TypeScript 严格模式检查
